# 安装ZSH
## 使用自动化脚本来帮助你安装zsh以及zsh配置
主要为ubuntu/debian来安装
其余系统可以修改`auto_install.sh`脚本上2和3行来安装，毕竟那两行只是用来安装zsh而已，后面的都一样，你可以删掉或者改成你系统安装zsh的方法。
**确保脚本文件在你终端的路径上**
运行:
```bash
chmod +x auto_install.sh
./auto_install.sh
```
# 测试情况
**经过测试，如果需要完全运行成功的话，需要执行这个脚本两次**
然后会有一个引导配置选项，这个不是我写的，是zsh官方自带的。
然后你需要手动运行`chsh -s $(which zsh)`,如果终端没有生效的话。当然运行后记得logout(注销或者重启)
